============================================================
 LocationCircle — Requirements & Dependencies
============================================================
NOTE: LocationCircle is a Node.js + TypeScript project.
Runtime dependencies are managed with npm (see client/package.json
and server/package.json) — NOT pip. This file documents the
prerequisites and the dependency list for reference/submission.

Install everything with:
    cd client && npm install
    cd ../server && npm install

------------------------------------------------------------
 SYSTEM PREREQUISITES
------------------------------------------------------------
Node.js            >= 20.x
npm                >= 10.x
Git                any recent version
Docker + Compose   (optional) for local PostgreSQL + Redis
PostgreSQL         >= 15   (local, or Neon cloud)
Redis              >= 7    (local, or Upstash cloud)

------------------------------------------------------------
 CLOUD ACCOUNTS / EXTERNAL SERVICES
------------------------------------------------------------
Firebase                 Google sign-in + Cloud Storage (required)
Neon                     managed PostgreSQL            (deploy)
Upstash                  managed Redis                 (deploy)
Render                   API + WebSocket hosting       (deploy)
Vercel                   static PWA hosting            (deploy)
Google Maps Platform     Distance Matrix + Geocoding   (optional)
Meta WhatsApp Cloud API  WhatsApp bot                  (optional)
